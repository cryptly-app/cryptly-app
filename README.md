# 🔐 Cryptly App

Cryptly is a private and secure messaging app with:

- 🔒 Encrypted chat and voice calls
- 🧊 Private call logs
- 🧠 Multi-passcode access
- 🛡️ Screenshot protection
- 📝 Secure notes
- 📶 Offline access support

> Built using Kotlin + Jetpack Compose.
